# PageRank-Graph-Implementation


Adjacency list graph implementation and
PageRank algoritm giving importance score to each page


Input format

(number of pages) (iterations of pagerank algorithm)

(page from)  (page to)


Sample Input


11 1
google.com gmail.com
google.com maps.com
facebook.com ufl.edu
ufl.edu google.com
ufl.edu gmail.com
maps.com facebook.com
maps.com quora.com
quoa.co maps.edu
quora.com ufl.com
quora.com facebok.co.in
quora.com quoa.co


Output for Sample
facebok.co.in 0.10
facebook.com 0.10
gmail.com 0.10
google.com 0.10
maps.com 0.10
maps.edu 0.10
quoa.co 0.10
quora.com 0.10
ufl.com 0.10
ufl.edu 0.10
